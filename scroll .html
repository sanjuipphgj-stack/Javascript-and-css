JavaScript Basics

JavaScript is a versatile programming language that enables interactive web pages. It's an essential part of web development.

// Simple JavaScript example function greetUser(name) { return `Hello, ${name}! Welcome to our website.`; } console.log(greetUser("Alex"));

JavaScript can manipulate HTML content, handle events, create animations, and much more.

CSS Fundamentals

CSS (Cascading Style Sheets) is used to style and layout web pages. With CSS, you can control the color, font, spacing, and positioning of elements.

/* CSS example for styling buttons */ .btn { background: linear-gradient(45deg, #1a2a6c, #b21f1f); color: white; padding: 12px 24px; border: none; border-radius: 5px; cursor: pointer; transition: all 0.3s ease; } .btn:hover { transform: translateY(-3px); box-shadow: 0 10px 20px rgba(0, 0, 0, 0.2); }

Modern CSS includes features like Flexbox, Grid, animations, and responsive design techniques.

Scroll Effects

Scroll effects can enhance user experience by creating engaging animations as users navigate your page.

// JavaScript for scroll animations window.addEventListener('scroll', function() { const sections = document.querySelectorAll('.section'); sections.forEach(section => { const sectionTop = section.getBoundingClientRect().top; const windowHeight = window.innerHeight; if (sectionTop < windowHeight * 0.85) { section.classList.add('visible'); } }); });

You can create parallax effects, fade-in animations, and more using scroll events in JavaScript combined with CSS transitions.
